  • Information Retrieval System add
    • Description: An information retrieval system to retrieve relevant documents from a corpus based on the user's search query. The system uses Latent Semantic Indexing as its underlying technology to extract concept of a document and then perform cosine similarity measures for the search query against each document to rank them by relevance. The sytem uses Machine Learning papers downloaded from ArXiv as its corpus.
    • My Work:
      • Downloaded latest Machine Learning reserach papers from ArXiv using ArXiv's API
      • Carried out data pre-processing
      • Built a TF-IDF and LSA pipeline
      • Trained the model on the pre-processed research paper documents
      • Built a web interface to query the model and retrieve relevant documents
      • Used cosine distance to rank the retrieved documents based on the relevance to search query
      • Documentation [Link]
    • Technologies used:
      Python 3.6
      Flask Framework, Python
      Scikit-learn
      HTML, CSS, JavaScript
